6. What is Header and Content Components in React Js?

Header Component

A Header component is usually the top section of a web page or application and contains elements like the logo, navigation links, search bars, and other important links or buttons. The Header component often remains constant across different pages of the application, providing a consistent navigation and branding experience.

Code:

            import React from 'react';

            const Header = () => {
              return (
                <header>
                    <nav>
                        <ul>
                            <li><a href="/">Home</a></li>
                            <li><a href="/about">About</a></li>
                            <li><a href="/contact">Contact</a></li>
                        </ul>
                    </nav>
                </header>
              );
            };

            export default Header;

Content Component

A Content component typically refers to the main area of the page where the primary information or features are displayed. This could include articles, data tables, forms, or any other type of content that varies from page to page within the application.

Code:

            import React from 'react';

            const Content = ({ children }) => {
              return (
                <main>
                  {children}
                </main>
              );
            };

            export default Content;

Usage

These components can be used together in a parent component to form a structured page layout. For example:

Code:

                import React from 'react';
                import Header from './Header';
                import Content from './Content';

                const App = () => {
                  return (
                    <div>
                        <Header />
                            <Content>
                                <h1>Welcome to My Website
                                <p>This is the main content of the page.

                            </Content>
                        </div>
                  );
                };

                export default App;
