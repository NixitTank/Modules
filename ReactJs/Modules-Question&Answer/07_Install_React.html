7. How to install React Js on Windows, Linux Operating System? How to Install NPM and How to check version of NPM?

ChatGPT Installing React.js on both Windows and Linux involves setting up Node.js and npm (Node Package Manager), as React is built upon these technologies. Here’s a step-by-step guide for both operating systems:

Installing React.js on Windows

1. Install Node.js and npm:

  • Download the Node.js installer from the official Node.js website.
  • Run the installer and follow the instructions. The installer will also install npm automatically.
  • After installation, open Command Prompt (CMD) and verify the installation: 

    •             node -v
            npm -v

    2. Create a React App:

  • Open Command Prompt. 

    •         npx create-react-app my-app

  • Navigate into your project directory: 

    •         cd my-app

  • Start the development server: 

    •         npm start


    Installing React.js on Linux

    1. Install Node.js and npm:

  • Update your package index: 

    •         sudo apt update

  • Install Node.js and npm. For the latest version, use NodeSource: 

    •         sudo apt install -y nodejs

  • Verify the installation: 

    •         node -v
        npm -v

    2. Create a React App:

  • Open a terminal.
  • Use npx to create a new React app: 

    •         npx create-react-app my-app

  • Navigate into your project directory: 

    •         cd my-app

  • Start the development server: 

    •         npm start

    Installing npm and Checking npm Version

    If npm is not installed or you need to update it, you can follow these steps:

    1. Install npm:

  • On Windows, npm comes with Node.js installation.
  • On Linux, if npm is not installed, you can install it using: 

    •         sudo apt install npm

    2. Update npm to the latest version: 

            npm install -g npm@latest

    3. Check the npm version: 

            npm -v

    Summary

    • Windows:
      • Install Node.js and npm from the official website.
      • Use npx create-react-app my-app to set up a new React app.
      • Start the development server with npm start.
    • Linux:
      • Use curl to get the latest Node.js setup script and install Node.js and npm.
      • Use npx create-react-app my-app to set up a new React app.
      • Start the development server with npm start.
    • Check npm version:
      • Use npm -v to check the installed version of npm.
