5. What is Components in React Js?

Components are the building blocks of React applications. They are reusable pieces of code that can be used to create user interfaces. Components can be simple, like a button or a text input, or they can be more complex, like a navigation bar or a photo gallery.

Components are independent and reusable bits of code. They serve the same purpose as JavaScript functions, but work in isolation and return HTML via a render() function. Components come in two types, Class components and Function components.

Class components are defined using the class keyword. They must include a render() method that returns HTML.

Function components are defined using a regular JavaScript function. They also require a render() method that returns HTML.

Components can be nested inside of other components. This allows you to create complex user interfaces out of smaller, more manageable pieces.


Here is an example of a simple React component: 

            function Welcome(props) {
                return <h1>Hello, {props.name}!</h1>;
              }

This component takes a name prop and returns an h1 element with the text "Hello, {props.name}!".

Components can also have state. State is data that is specific to a component and can change over time. Components can use state to keep track of things like the value of a text input or the selected item in a list.


Here is an example of a React component with state: 


            class Counter extends React.Component {
                state = {
                  count: 0
                };
              
                handleClick() {
                  this.setState({ count: this.state.count + 1 });
                }
              
                render() {
                  return (
                    <div>
                        <h1>Count: {this.state.count}</h1>
                        <button onClick={this.handleClick}>Increment</button>
                    </div>
                  );
                }
              }

This component has a state object that tracks the current count. The handleClick() method increments the count and the render() method displays the current count.

Components are a powerful tool for creating reusable and maintainable user interfaces. They are the foundation of React applications.
