8. How to check version of React Js?

To check the version of React JS that you are using in your project, you can follow several methods depending on your setup:

Method 1: Check package.json

  • Open your project directory.
  • Locate the package.json file.
  • Look for the react and react-dom dependencies to see their versions. 

    •             {
                "dependencies": {
                  "react": "^17.0.2",
                  "react-dom": "^17.0.2"
                }
              }

    Method 2: Using npm list

  • Open a terminal or command prompt in your project directory.
  • Run the following command to check the version of React installed: 

    •             npm list react

    This will display the version of React along with other dependency information.

    Method 3: Using yarn list

    If you are using Yarn as your package manager, you can use a similar command:

  • Open a terminal or command prompt in your project directory.
  • Run the following command: 

    •             yarn list --pattern react

    This will list all packages matching the pattern react and their versions.
