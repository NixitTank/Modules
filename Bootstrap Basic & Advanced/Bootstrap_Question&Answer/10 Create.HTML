10 What is a bootstrap card and how would you create one?

A basic Bootstrap card consists of a . card class with the . card-body class, allowing you to add text, images, links, and other Bootstrap elements within a rounded border.

Example :

        <div class="container">
            <div class="card" style="width:400px">
              <img class="card-img-top" src="D:\TOPS-WORK\Modules\Bootstrap Basic & Advanced\Img\microsoft-storage.jpg.imgo.jpg" alt="Card image" style="width:100%">
              <div class="card-body">
                <h4 class="card-title">John Doe</h4>
                <p class="card-text">Some example text some example text. John Doe is an architect and engineer</p>
                <a href="#" class="btn btn-primary">See Profile</a>
              </div>
            </div>
            <br>
        </div>

Output :

Microsoft

Microsoft is the largest vendor of computer software in the world. It is also a leading provider of cloud computing.

See Profile
