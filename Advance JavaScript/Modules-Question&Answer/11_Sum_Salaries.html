11. Sum the properties There is a salaries object with arbitrary number of salaries. Write the function sumSalaries(salaries) that returns the sum of all salaries using Object.values and the for..of loop.If salaries is empty, then the result must be 0. 

            let salaries = {
                "John": 100,
                "Pete": 300,
                "Mary": 250
                };
                alert( sumSalaries(salaries) ); // 650

Code:


                <div>
                    <h1>Sum of Salaries</h1>
                    <p>The sum of salaries is: <span id="result"></span></p>
                </div>

                <script>
                    function sumSalaries(salaries) {
                      let total = 0;
                      for (let salary of Object.values(salaries)) {
                        total += salary;
                      }
                      return total;
                    }
                
                    let salaries = {
                      "John": 100,
                      "Pete": 300,
                      "Mary": 250
                    };
                
                    document.addEventListener('DOMContentLoaded', function() {
                    document.getElementById('result').textContent = sumSalaries(salaries);
                    });
                </script>
