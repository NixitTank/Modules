7. Write the code, one line for each action:

        a) Create an empty object user.
        b) Add the property name with the value John.
        c) Add the property surname with the value Smith.
        d) Change the value of the name to Pete.
        e) Remove the property name from the object.

Check the console for results


            <script>
                // a) Create an empty object user.
                let user = {};
        
                // b) Add the property name with the value John.
                user.name = "John";
                console.log("After adding name:", user);
        
                // c) Add the property surname with the value Smith.
                user.surname = "Smith";
                console.log("After adding surname:", user);
        
                // d) Change the value of the name to Pete.
                user.name = "Pete";
                console.log("After changing name:", user);
        
                // e) Remove the property name from the object.
                delete user.name;
                console.log("After removing name:", user);
            </script>

Output:

            After adding name: {name: "John"}
            After adding surname: {name: "John", surname: "Smith"}
            After changing name: {name: "Pete", surname: "Smith"}
            After removing name: {surname: "Smith"}
