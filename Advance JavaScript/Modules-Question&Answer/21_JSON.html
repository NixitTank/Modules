21. What is JSON

JSON (JavaScript Object Notation) is a lightweight data interchange format that is easy for humans to read and write and easy for machines to parse and generate. It is commonly used for transmitting data in web applications (e.g., sending data from a server to a client so it can be displayed on a web page).

Key Features of JSON:

1. Text Format: JSON is a plain text format that uses a structure similar to JavaScript object literals.

2. Data Structure: It is based on two structures:

  • A collection of name/value pairs: In various languages, this is realized as an object, record, struct, dictionary, hash table, keyed list, or associative array.
  • An ordered list of values: In most languages, this is realized as an array, vector, list, or sequence.
    • 3. Syntax Rules:
  • Objects: Represented as name/value pairs enclosed in curly braces {}. Each name is followed by a colon : and the name/value pairs are separated by commas. 
    •                 {
                    "name": "John",
                    "age": 30,
                    "city": "New York"
                  }
  • Arrays: Ordered collections of values enclosed in square brackets [] and separated by commas. 
    •                 ["apple", "banana", "cherry"]
  • Values: Can be strings (in double quotes), numbers, objects, arrays, true, false, or null.
  • Strings: Must be enclosed in double quotes and can contain escape characters. 
    •                 "name": "John"
  • Numbers:Can be integers or floating-point numbers. 
    • "age": 30

    Use Cases:

    • Web APIs: JSON is commonly used to exchange data between a server and web application.
    • Configuration Files: Many software applications use JSON format for configuration settings.
    • Data Storage: Some NoSQL databases, like MongoDB, use a JSON-like format (BSON) to store data.

    Example:

    Here is a complete JSON example representing a person with a name, age, and list of favorite fruits: 

              
            {
                "name": "John",
                "age": 30,
                "isEmployed": true,
                "address": {
                  "street": "123 Main St",
                  "city": "New York",
                  "zipcode": "10001"
                },
                "favoriteFruits": ["apple", "banana", "cherry"]
              }
