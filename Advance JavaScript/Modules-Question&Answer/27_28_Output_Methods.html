28. How to used JavaScript Output method?

Explanation:

  1. console.log():This will log a message to the browser's console. Open the browser’s developer tools (F12) to see the console output.
  2. document.write(): This writes directly to the document. Note that using document.write() after the document is fully loaded can overwrite the entire page.
  3. innerHTML: This changes the content of the element with the ID output.
  4. alert():This will show an alert box with the specified message.
  5. confirm():This will show a confirmation box, and the result (OK or Cancel) will be logged to the console.
  6. prompt():This will show a prompt box asking for the user's name, and the input will be displayed on the document.

Code:


                <div id="output">This will be changed by JavaScript</div>

                <script>
                    // console.log() example
                    console.log("This is a message for the console.");

                    // document.write() example
                    document.write("This text is written by document.write().");

                    // innerHTML example
                    document.getElementById("output").innerHTML = "This text is set by innerHTML.";

                    // alert() example
                    alert("This is an alert box!");

                    // confirm() example
                    if (confirm("Do you like JavaScript?")) {
                        console.log("User likes JavaScript.");
                    } else {
                        console.log("User does not like JavaScript.");
                    }

                    // prompt() example
                    var userName = prompt("Please enter your name:", "Harry Potter");
                    if (userName != null) {
                        document.write("
Hello, " + userName + "!");
                    }
                </script>

Output:

This will be changed by JavaScript.
