59. Array vs object defences in JS?

In JavaScript, arrays and objects are both used to store collections of data, but they have distinct characteristics, use cases, and methods of interaction. Here’s a detailed comparison:

Arrays

  1. Definition:
    • Arrays are ordered collections of elements.
    • Elements are accessed by their numerical index.
  2. Syntax:
    • Defined using square brackets [].
    • Example: let fruits = ["apple", "banana", "cherry"];
  3. Accessing Elements:
    • By index: fruits[0] returns "apple".
  4. Length:
    • Arrays have a length property that returns the number of elements.
    • Example: fruits.length returns 3.
  5. Methods:
    • Common array methods include: push(), pop(), shift(), unshift(), slice(), splice(), concat(), map(), filter(), reduce(), etc.
    • Example: fruits.push("date") adds "date" to the end of the array.
  6. Use Cases:
    • Suitable for ordered collections of items.
    • Ideal for lists, sequences, and stacks/queues.

Objects

  1. Definition:
    • Objects are collections of key-value pairs (properties).
    • Keys (also called properties) are usually strings or symbols.
  2. Syntax:
    • Defined using curly braces {}.
    • Example: let person = { name: "John", age: 30, job: "developer" };
  3. Accessing Properties:
    • By key: person.name or person["name"] returns "John".
  4. Keys and Values:
    • The set of keys can be retrieved using Object.keys().
    • The set of values can be retrieved using Object.values().
  5. Methods:
    • Common object methods include: Object.keys(), Object.values(), Object.entries(), hasOwnProperty().
    • Example: Object.keys(person) returns ["name", "age", "job"].
  6. Use Cases:
    • Suitable for collections of named properties.
    • Ideal for representing entities with attributes (like a user, product, etc.).

Examples

Array Example: 

            let fruits = ["apple", "banana", "cherry"];
            console.log(fruits[1]); // Outputs: banana
            fruits.push("date");
            console.log(fruits); // Outputs: ["apple", "banana", "cherry", "date"]

Object Example: 

            let person = { name: "John", age: 30, job: "developer" };
            console.log(person.name); // Outputs: John
            person.age = 31;
            console.log(person); // Outputs: { name: "John", age: 31, job: "developer" }
