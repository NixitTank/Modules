53. Use lastIndexOf to determine the position of the last occurrence of a in 30 Days Of JavaScript?

Code:


                const str = "30 Days Of JavaScript";
                const position = str.lastIndexOf('a');

                if (position !== -1) {
                  console.log(`The last occurrence of 'a' is at position ${position}.`);
                } else {
                  console.log("The letter 'a' is not found in the string.");
                }

Output:


                The last occurrence of 'a' is at position 19.
