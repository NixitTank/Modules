58. What is Bom vs Dom in JS?

Aspect DOM (Document Object Model) BOM (Browser Object Model)
Purpose Manages and interacts with the structure, content, and style of web documents. Interacts with the browser window and provides control over browser features.
Structure Represents the HTML or XML document as a tree of objects. Consists of various browser-specific objects like window, navigator, location, history, and screen.
Standardization Standardized by the W3C. Not standardized; implementation can vary between browsers.
Scope Focuses on the document content and structure. Focuses on the browser environment and controls.
Key Features - Element Manipulation: Create, modify, and remove elements.<br> - Event Handling: Manage events like clicks, input, and form submissions.<br> - Dynamic Content: Update the document content and structure dynamically. - Window Control: Open, close, move, and resize browser windows.<br> - Browser Information: Access information about the browser and user environment.<br> - URL Manipulation: Change and retrieve the URL of the current document.<br> - History Management: Navigate through the browser history.
Primary Objects document, Element, Node, Event window, navigator, location, history, screen
Example Use Case Changing the text content of an HTML element when a button is clicked. Navigating back in the browser history or reloading the current page.
