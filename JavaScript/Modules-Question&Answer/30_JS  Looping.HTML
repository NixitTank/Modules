30. What are the looping structures in JavaScript? Any one Example?

for loop : A for loop is used to repeatedly execute a block of code based on a condition.

while loop : A while loop executes a block of code as long as a condition is met.

do-while loop : Similar to a while loop, a do-while loo.

Example :


            <h3 id="ans"></h3>

            <script>
                const cars = ["BMW", "Volvo", "Saab", "Ford", "Fiat", "Audi"];
        
                let text = "";
                for (let i = 0; i < cars.length; i++) {
                text += cars[i] + "<br>";
                }
        
                document.getElementById("ans").innerHTML = text;
            </script>
