4. Write a mul Function Which will Work Properly When invoked With Following Syntax.

The MUL function is a miniature of the multiplication function. In this function, we call the function that required an argument as a first number, and that function calls another function that required another argument and this step goes on. The first function’s argument is x, the second function`s argument is y and the third is z, so the return value will be xyz.

Syntax:


            <script> 
                function mul(x) { 
                return function(y) { 
                    return function(z) { 
                    return x*y*z; 
                    }; 
                } 
                } 
                
                console.log(mul(2)(3)(5)); 
                console.log(mul(2)(3)(4)); 
            </script >

Output:

30
24
